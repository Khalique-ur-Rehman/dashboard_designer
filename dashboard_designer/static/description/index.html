Dashboard Studio

Build powerful, beautiful dashboards inside Odoo. No coding required.

Dashboard Studio empowers Odoo users to create stunning analytics dashboards with KPI cards, interactive charts, and real-time business data. All from a simple interface. No developer needed.

Key Features

  • Create and manage multiple dashboards
  • Add KPI widgets with real-time values
  • Add text and annotation widgets
  • Line, Bar, Radar, Scatter, Gauge, Funnel, Heatmap and Sankey charts
  • Theme-aware dashboard preview
  • Export dashboard as PNG image
  • Printable PDF dashboard report
  • Dashboard home with summary tiles
  • AI-powered widget insights

Dashboard Builder

Easily create and manage multiple dashboards with interactive widgets and real-time business data.

KPI Cards and Analytics

Display important business metrics with beautiful KPI cards and visual charts updated in real time.

Dashboard Themes

Choose from multiple visual themes to fully customize the look and feel of your dashboards.

Export and Reports

Export dashboards as PNG images or generate printable PDF reports for easy sharing with your team.

Technical Details

  • Compatible with Odoo 18
  • Works with Odoo Community Edition
  • Built with modern OWL framework and JavaScript
