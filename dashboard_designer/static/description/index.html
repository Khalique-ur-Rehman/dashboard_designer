Dashboard Studio

Dashboard Studio allows users to create powerful and beautiful dashboards inside Odoo without any coding. Build KPI cards, charts, and custom dashboards to visualize business data in a modern way.

Key Features

  • Create multiple dashboards
  • Add KPI widgets
  • Add text widgets
  • Create charts and visual analytics
  • Theme-aware dashboard preview
  • Export dashboard as PNG
  • Printable dashboard report (PDF)
  • Dashboard home with summary tiles

Dashboard Builder

Users can easily create and manage multiple dashboards with interactive widgets and real-time data.

KPI Cards & Analytics

Display important business metrics with beautiful KPI cards and visual charts.

Dashboard Themes

Choose from different visual themes to customize the dashboard appearance.

Export & Reports

Dashboards can be exported as PNG images or printed as PDF reports for easy sharing.

Technical Details

  • Compatible with Odoo 18
  • Works with Odoo Community Edition
  • Built using modern OWL and JavaScript
